# AI Virtual Interview Evaluator

## Overview
This project evaluates a candidate's confidence during a mock interview using computer vision techniques.  
The system analyzes facial expressions, eye contact, and posture in real time and generates a confidence score.  
A Streamlit interface is used to make the system interactive and easy to use.

## Features
- Emotion detection
- Eye contact detection
- Posture analysis
- Real time Confidence score
- Final Interview report

## Techstack
- Python
- streamlit
- OpenCV
- MediaPipe
- Deep Learning

## How to Run the Project
1. Clone the repository
2. Navigate to the project folder
3. Run the following command:

streamlit run app.py
